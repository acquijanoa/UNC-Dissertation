There are one pdf file and six R files in the folder:

* "Blinded_Supplement.pdf" is the blinded Supplementary Material for review.

* "Simulation_1_functions.R" contains the auxiliary functions used to conduct the simulation in Section 6.1.

* "Simulation_1_test_code.R" contains a simple test code, and the code used to generate Table 1 of the manuscript (change the number of bootstrap iterations to get different results for the proposed method).

* "Simulation_2_functions.R" contains the auxiliary functions used to conduct the simulation in Section 6.2.

* "Simulation_2_test_code.R" contains a simple test code, and the code used to generate Table 2 of the manuscript (change the number of bootstrap iterations to get different results for the proposed method).

* "Simulation_3_functions.R" contains the auxiliary functions used to conduct the simulation in Section S9 of the Supplementary Material.

* "Simulation_3_test_code.R" contains a simple test code, and the code used to generate Table S1 of the Supplementary Material (change the number of bootstrap iterations to get different results for the proposed method).
